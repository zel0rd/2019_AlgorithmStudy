## Programmers_file 제목 작성 요령
### CODING_TEST
- [ 문제_작성자.확장자 ] 형식으로 만든다.
- 문제 사이의 공백은 _ 으로 표시한다.
- 문제와 작성자 사이에도 _ 을 추가한다.
### SKILL_CHECK
- [ LEVEL\d-\d_작성자.확장자 ] 형식으로 만든다
- 문제 사이의 공백은 _ 으로 표시한다.
- 문제와 작성자 사이에도 _ 을 추가한다.
